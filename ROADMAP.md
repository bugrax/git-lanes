# Roadmap

## v0.1.0 (Current)

- [x] Session lifecycle (start, end, abort)
- [x] Change tracking (track, commit, undo, squash)
- [x] Conflict detection with resolution suggestions
- [x] Test runner with auto-detection
- [x] Multi-forge PR support (GitHub, GitLab, Bitbucket)
- [x] Adapter hooks (Claude Code, Cursor, Aider)
- [x] File locking for manifest safety
- [x] Configurable branch prefix

## v0.2.0

- [ ] Watch mode for real-time conflict notification
- [ ] Session health monitoring daemon
- [ ] Improved auto-merge with three-way diff
- [ ] Session templates for common workflows
- [ ] `git lanes stash` for temporary session parking

## v0.3.0

- [ ] Web dashboard for session visualization
- [ ] Webhook notifications (Slack, Discord)
- [ ] Session metrics and analytics
- [ ] Cross-repository session support
- [ ] Plugin system for custom adapters

## v1.0.0

- [ ] Stable API with semver guarantees
- [ ] Comprehensive documentation site
- [ ] Windows native support (without WSL)
- [ ] Performance benchmarks and optimization
- [ ] Security audit
